## 🚀 Overview

> A fully interactive **Dijkstra Algorithm Visualizer** that demonstrates shortest path finding in real-time using a grid-based UI.

This project was **refactored from React to EJS + Express**, showcasing strong understanding of:
- Frontend architecture migration
- DOM-based rendering
- Algorithm visualization

---

|-------------------|----------------|

| No architecture thinking | ✔ React → EJS migration |
